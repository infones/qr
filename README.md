QR Code generator
=================

Usage
-----

http://qr.edocu.sk/?data=http://example.com

### Parameters:

* **data  - content**
* level - error correction level ("L"ow,"M"edium,"Q"uarter,"H"igh) (*default: H*)
* size  - pixels per point (*default: 5*)
* border - border size in points (*default: 2*)
